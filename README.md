## Incialização

Para inicializar o projeto via comando, especifique a versão do dotnet a ser utilizada.
A versões suportadas são: 
- net8.0
- net9.0
- net10.0

exemplo:
```bash
dotnet run --framework net10.0 Program.cs
```
